## 项目愿景

打造一个以专利为核心锚点的多维知识管理系统：

- **Excel的灵活** + **数据库的严谨** + **AI的智能**
- **全生命周期管理**：从提案到失效的完整旅程
- **全维度覆盖**：几百个自定义字段，随时新增
- **跨国界支持**：PCT、巴黎公约、多法域、多语言
- **知识网络**：引用、同族、双向链接、图谱可视化
- **基建属性**：API优先，可与其他系统打通

---

## 核心原则

1. **开箱即用**：双击安装，不需要装Python、数据库等任何环境
2. **本地优先**：数据存在本地，隐私安全有保障
3. **Excel习惯**：操作方式尽量和Excel一致，降低学习成本
4. **配置优于编码**：业务规则尽量可配置，减少硬编码
5. **渐进式增强**：核心功能先做扎实，高级功能逐步添加
6. **AI是助手**：AI永远是辅助，最终决定权在用户
